Who is that guy?

I'm a German, self-taught web developer with a strong focus and enthusiasm in JavaScript (especially ES6) as well as other emerging technologies from the frontend stack. I work as User Interface Developer at Razorfish Germany and live in Berlin.

I love the web and everything about it. Since I started working as a web developer, I had the privilege to work with companies such as Audi, Here (Nokia), Coca-Cola, Axa, Deutsche Bahn, McDonald's, Demodern & more. Most of my knowledge is based on the open-source work of other developer. Hence I want to give something in return and decided to start blogging myself.

In my spare time I like to fiddle around with different technologies and contribute to as well as work on my own open source projects:

  • FrontBook A small and modern frontend boilerplate, enabling you to write ES6 today in production-ready projects.
  • BrowDash A fully customisable dashboard Chrome extension, written with native Web Components and ES6.
